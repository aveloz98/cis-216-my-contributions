Why the end of Windows 10 will be significant

My experiences with Windows 10

I have been a user of Windows 10 since it was released back in 2015. Since then, I have followed its development through each biannual feature update that was released for it & followed up on any issues that arose after it was released. With Windows 11, and soon Windows 12, as the latest OS & with Windows 10's support down to jsut a few more months. I heard about why many people are choosing not to go to Windows 11.

Reasons why businesses & indivuduals refuse to update from Windows 10

  1. Many of these Windows 10 machines don't meet the crazy system requirements for this new OS. Specifically, many of them don't have the TPM 2.0 Chip or Secure Boot on their motherboards. I was very lucky to have a Dell laptop from 2015 that met almost every requirement for Windows 11, except for the processor since it has a 5th generation i5. There are methods out there, such as Rufus, to remove the requirements and be able to install Windows 11 on officially unsupported devices, as long as they have at least a TPM 1.2 chip. Microsoft however does not recommend doing so because of compatibility problems with drivers that could occur, along with them possibly stopping all security updates on these machines in the future.
  2. People prefer Windows 10 over Windows 11 as that is the best OS they could run on their current machine.
  3. Businesses can pay for up to 3 more years of updates just like with Windows 7, however individuals only get one year of additional updates.
  4. With the kind of software businesses use for their everyday operations, they can only run it on a certain OS. Many businesses are still using outdated OS's because they don't have to money to ensure their software can correctly run on the newest version of Windows.
  5. People don't have the money to go out and buy a new computer, especially the new Copilot+ PC's that Microsoft debuted in June of last year. Many of these machines now cost as much as $2,000, which is insane.
